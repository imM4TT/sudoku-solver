# Solver de Sudoku
- Le programme fonctionne avec un ficher texte en entrée  
- Il résout tout type de Sudoku 9*9 avec n'importe quels caractères (chiffres, lettres, symboles)

# Exemple de fichier d'entrée
```
C_G____A_
___H_E__F
I___G___D
__H_E____
B__F_H__C
____I_E__
H___A___G
F__I_D___
_C____I_H
```
